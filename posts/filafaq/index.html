Islamic database website launched

Written by Zill_Laiss on 2026-01-28
filafaqgo
بسم الله

Alhamdu lillāh, after more than a year of development, I have launched to public the first version of Fī 'l-āfāq (aka. filafaq), an Islamic database website on November 15th 2025.

The goal is to allow people, especially Muslims, to discover information and beneficial knowledge from across the horizons, while also preserving the knowledge with the collaborations of Muslims so it is not gone with the passage of time. The main entry is Literature, which includes book, article, and any beneficial written form of Islamic knowledge.

The site is backend written in Go. The frontend uses Templ (Go library), HTMX, and Alpine as the main components.

It's been 2 months since then, and biidhnillāh, I'm working on populating the website with information. There's currently only less than 20 literatures, and I'm planning to add more.

The site is still pretty much in early launch so the entries count overall is still few. The features are also basics, but more of them to come, in shāa Allah.

Current features

  • Free registration.
  • Extensive information regarding literature and its editions.
  • Add your own favourite literature to the database.
  • Add quranic reference to a literature.
  • See relation for literature to other literatures.
  • Mark literature with Tag for easier discoverability.
  • Discover random literature.
  • Add star to your favorite literatures.
  • Revert back any accidental change with reversion feature.
  • Discuss literature (and other Islamic topics) with other users.
  • User's literature listing.

Planned features

  • Advanced search.
  • Light mode.
  • Better mobile support.
  • Text hosting for public domain literatures.
  • archive.org-style books borrowing
  • Links to libraries & stores
  • Opening the database.
  • Create API for developers to use.
  • Export entry information as printable media (odt, docx, etc.)
  • Adding more entry types (Applications, book references, etc.).

Known issues

  • No favicon yet.
  • There's no way to mark books with volumes together (Literature Group entry is coming).
  • Documentation is incomplete.
  • Inconsistent visual across the sites.
